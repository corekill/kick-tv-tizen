Soukromí bez drobných písmen.

Kick TV nepoužívá běžnou telemetrii. Počítá pouze anonymní první spuštění aplikace.

Co aplikace odešle

Po prvním úspěšném spuštění odešle jeden prázdný požadavek na corekill.cz. Neobsahuje ID instalace, model televize, jazyk, hledání, sledované kanály, chat ani jiné údaje o používání.

Stejně jako u každého HTTPS spojení projde síťová adresa přes webový server a reverzní proxy. Samotný přijímač ji ale nepřebírá, neukládá ani nespojuje s výsledným počtem.

Co se uloží

  • Na serveru se pouze zvýší jedno společné číslo celkových instalací.
  • Přijímač nevytváří záznamy o jednotlivých požadavcích; jeho backendová cesta má access log vypnutý.
  • V televizi se uloží pouze příznak kicktv.installReported, aby se signál neposílal znovu.

Co číslo znamená

Jde o počet prvních úspěšných spuštění od verze 2.2. Odinstalování aplikace počet nesníží. Nová instalace nebo smazání dat aplikace se může započítat znovu.

Privacy without fine print.

Kick TV does not use regular telemetry. It only counts anonymous first launches.

What is sent

After the first successful launch, the app sends one empty request to corekill.cz. It contains no installation ID, TV model, language, searches, watched channels, chat, or other usage data.

As with every HTTPS connection, the network address passes through the web server and reverse proxy. The counter receiver itself does not accept, store, or associate it with the resulting total.

What is stored

  • The server only increments one shared aggregate installation total.
  • The receiver creates no per-request records, and its backend route has access logging disabled.
  • The TV stores only the kicktv.installReported flag so the signal is not sent again.

What the number means

It is the number of first successful launches since version 2.2. Uninstalling the app does not decrease it. A reinstall or clearing the app data may be counted again.
